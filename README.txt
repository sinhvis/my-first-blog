How to set up a virtual environment?
This is really important if need to set up Django or any other Python
package in a localized environment that doesn't affect the rest of the
installation.
Following instructions are specific for Linux.
First make a new directory that will hold the new project:
mkdir newDir
cd newDir
To make a virtual environment called myenv:
python3 -m venv myenv
